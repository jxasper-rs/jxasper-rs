# jxasper-rs

**jxasper-rs** is a Rust-based tool for generating JasperReports JRXML
from Excel templates.

The goal is to allow report layouts to be designed in Excel and
converted into deterministic, version-controlled JRXML files.

## Status

Early development (v0).

## Scope (v0)

- Excel (.xlsx) parsing
- Layout extraction
- JRXML generation
- No UI guarantees
- No preview rendering

## Non-Goals

- Replacing Jaspersoft Studio
- Supporting all JasperReports features
- Acting as a visual report designer
